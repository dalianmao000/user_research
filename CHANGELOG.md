## [1.0.0] - 2026-05-21

### Added
- Initial project structure
- Docker Compose configuration (n8n, Weaviate, PostgreSQL)
- Obsidian Vault templates
- Claude Code Skills for user research workflow
  - `/ur-start` - Project initialization
  - `/ur-interview` - Interview analysis
  - `/ur-requirements` - PRD generation
  - `/ur-usability` - Usability testing assistance
  - `/ur-competitive` - Competitive analysis
  - `/ur-report` - Report generation
  - `/ur-insights` - Cross-project insights
- AI Agents
  - Data Fusion Agent - Interview analysis
  - Full-Cycle UR Agent - PRD generation
  - Global Insight Agent - Report generation
  - Process Orchestrate Agent - Workflow coordination
  - Compliance Agent - Privacy and compliance
- Prompt templates for common tasks
- Obsidian sync tool
- GitHub Actions CI workflow
